# Bingoplay PWA

Jogo de Bingo Offline-First para iOS e Android.

## 🚀 Funcionalidades

- ✅ 100% Offline - Funciona sem internet
- ✅ Persistência local - Não perde o estado
- ✅ PWA instalável no iOS/Android
- ✅ Interface mobile-first

## 📱 Como usar

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build
```

## 🖥️ PWA

Para instalar no iOS:
1. Abra no Safari
2. Toque em "Compartilhar"
3. Selecione "Adicionar à Tela de Início"

## 📋 Tecnologias

- React 18
- TypeScript
- Vite
- TailwindCSS
- Dexie.js (IndexedDB)
- Framer Motion
- Vite PWA Plugin
